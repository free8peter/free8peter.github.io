python class 以及static method

创建一个文件 Common.py 里面写入下面的

        class AllPaths:
            @staticmethod
            def album_work_directory(album_directory):
                album_path = "one path"
                if len(album_path) > 0:
                    album_path = album_directory + album_path
                    return album_path
                else:
                    print("需要传入bbabox绝对路径")
                    return ""

也就是静态方法要以@staticmethod开头。

其它文件中如何引入呢

from Common import AllPaths

work_paths = AllPaths()

关于python的静态方法：如果python一个函数里面没有self,那么可能会按静态方法执行，因为编译器认为函数里面如果没有self 那么可能是静态方法
