iOS memory dive

iOS 内存使用的工具
VMMAP
具体在命令行里面输入，即可，怎么用。
1.找pid progress id
首先跑起来Xcode, 之后点击Debug Memory Graph,之后File里export Memory Graph,这个时候可以看到文件里有一个中括号，中括号里面是progress id.
2.可以在打开苹果的原生应用活动监视器，之后可以找运行的应用的名称，后面的就是progress id.
3.终端里运行命令即可。vmmap --summary 49504
其中49504是progress id.
注意，这个进程必须是运行中的进程。
VIRTUAL means not real, means request, but not nessary using.
