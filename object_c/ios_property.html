iOS 里面属性

@property(nonatomic, getter=gogoName, copy) NSString *name;
@property(nonatomic, setter=gogoAge:, copy) NSString *age;
- (NSString *)gogoName {
return _name;
}
- (void)gogoAge:(NSString *)age {
// self.age = age; // 会发生死循环
_age = age;
}
属性的括号里写，getter=...; setter=...相当于指定了setter和getter方法，写self.name= 的时候，原来的setter 方法不再生效。相当于替换了原来的getter setter方法
